# Se controla! 🛒

O **Se controla!** é uma aplicação web progressiva e responsiva desenvolvida para auxiliar no controle financeiro em tempo real durante as compras. O projeto foi concebido para resolver o problema comum de perder a conta dos gastos antes de chegar ao caixa, permitindo uma gestão precisa do orçamento disponível.

---

## 🚀 Funcionalidades

- **Controle de Orçamento Dinâmico**: O usuário define um limite máximo de gastos que é persistido no navegador.
- **Cálculo Automático**: Soma instantânea de quantidade × preço unitário para cada item.
- **Feedback Visual Inteligente**: O card de status utiliza cores semânticas (Verde para dentro do limite, Vermelho para excedido) com alertas de valor ultrapassado.
- **Persistência de Dados**: Utiliza `LocalStorage` para que a lista e o limite não sejam perdidos ao fechar a aba ou o navegador.
- **Interface Premium**: Design moderno focado em UX (User Experience), com tipografia refinada e ícones escaláveis (SVG).

## 🛠️ Tecnologias Utilizadas

Este projeto foi construído utilizando as tecnologias fundamentais do desenvolvimento web moderno:

- **HTML5**: Estruturação semântica e acessível.
- **CSS3**: Estilização avançada com Variáveis CSS, Flexbox e animações de feedback.
- **JavaScript (ES6+)**: Manipulação dinâmica do DOM, lógica de algoritmos e persistência local.
